# pgdm
PostgreSQL Data Management Tool

# Install

## Requirements

 - [NodeJS](https://nodejs.org/en/)

## Install

```bash
npm install -g @ucd-lib/pgdm
```

# Usage

Help commands are descriped in CLI

```bash
pgdm --help
pgdm [command] --help
```
